## 📖 Project Overview
This is a responsive frontend web application built using React and Tailwind CSS.  
It includes authentication features like registration, OTP verification, and login using API integration.

---

## ✨ Features

- 🔐 User Registration
- 📩 OTP Verification
- 🔑 Login System
- 🔁 Resend OTP
- 👁️ Show/Hide Password
- 📱 Fully Responsive Design
- ⚡ Clean UI with Tailwind CSS

---

## 🛠️ Technologies Used

- React (Vite)
- Tailwind CSS
- Axios
- React Router DOM
