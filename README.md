## 📖 Project Overview
This is a responsive frontend web application built using React and Tailwind CSS.  
It includes authentication features like registration, OTP verification, and login using API integration.

---

## ✨ Features

- 🔐 User Registration
- 📩 OTP Verification
- 🔑 Login System
- 🔁 Resend OTP
- 👁️ Show/Hide Password
- 📱 Fully Responsive Design
- ⚡ Clean UI with Tailwind CSS

---

⚠️ OTP Note

The OTP is not sent via email in this testing environment.
After registration, the OTP can be found in the browser's developer tools (Network/Console tab) as part of the API response.

Please use that OTP to complete the verification process.

## 🛠️ Technologies Used

- React (Vite)
- Tailwind CSS
- Axios
- React Router DOM
